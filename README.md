WEATHERWEB-3.0 API
Made by Piotr Grosz

***********
HOW TO RUN?
***********

Click on link below:

    https://desheger.github.io/WeatherWeb-3.0/

OR 

Clone repository

    git clone https://github.com/DeSheger/WeatherWeb-3.0

Then go to cloned repository

    cd WeatherWeb-3.0

Install packages

    npm install

And run project!!!

    npm start

*******************************************
